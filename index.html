Maps & Sets


Question 1 : You are given a string (STR) of length N, consisting of only the lower case English alphabet. Your task is to remove all the duplicate occurrences of characters in the string.

Answer:

let s="abcadeecfb";
let uniqueCharacters=new Set();
for(i=0;i < s.length;i++){
  uniqueCharacters.add(s[i]);
}
result="";
for(let xyz of uniqueCharacters.keys()){
result=result+xyz;
}
console.log(result);

Question 2 : You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps)

Answer:

let s="abcadeecfb";
let myMap = new Map();
for(let i=0;i < s.length;i++){
  if(myMap.has(s[i])){
    let previousCount=myMap.get(s[i]);
    myMap.set(s[i],previousCount+1);
  }
  else{
    myMap.set(s[i],1);
  }

}
for(let [ch,freq] of myMap.entries()){
 console.log(ch+"="+freq);
}
